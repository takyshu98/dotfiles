# tmux キーバインド マニュアル

## プレフィックスキー

- **`Ctrl+z`** - プレフィックスキー（デフォルトの`Ctrl+b`から変更）

## 設定ファイルの操作

| キー | 説明 |
|------|------|
| `prefix` + `r` | 設定ファイルをリロード |
| `prefix` + `Ctrl+r` | 設定ファイルをリロード |

## ウィンドウ分割

| キー | 説明 |
|------|------|
| `prefix` + `\|` | ウィンドウを水平に分割 |
| `prefix` + `-` | ウィンドウを垂直に分割 |

## ペインのリサイズ（連続入力可能）

| キー | 説明 |
|------|------|
| `prefix` + `H` | ペインを左に5px拡大 |
| `prefix` + `J` | ペインを下に5px拡大 |
| `prefix` + `K` | ペインを上に5px拡大 |
| `prefix` + `L` | ペインを右に5px拡大 |

※ これらのキーは`repeat-time`（2000ms）の間、プレフィックスなしで連続入力可能

## ペインの移動

| キー | 説明 |
|------|------|
| `prefix` + `h` | 左のペインに移動 |
| `prefix` + `j` | 下のペインに移動 |
| `prefix` + `k` | 上のペインに移動 |
| `prefix` + `l` | 右のペインに移動 |
| `Ctrl+]` | 次のペインに移動（プレフィックス不要） |

## ペインを閉じる

| キー | 説明 |
|------|------|
| `prefix` + `x` | ペインを閉じる（確認プロンプトあり） |
| `Ctrl+d` または `exit` | シェルを終了してペインを閉じる |

## マウス操作

- マウス操作が有効化されています
- コピーモードでの動作:
  - `Enter`: 選択範囲をコピーしてコピーモード終了
  - マウスドラッグ終了時の自動コピーは無効化

## その他の設定

- **Escapeキー**: レスポンス時間を0msに設定（高速化）
- **ペイン番号**: 1から開始（デフォルトは0）
- **ステータスバー**: 1秒ごとに更新、中央揃え
